# Fake News Detection AI Project

This project uses Machine Learning to classify news as REAL or FAKE.

## Features
- Takes user input
- Predicts whether news is real or fake
- Uses Naive Bayes algorithm

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
1. Install libraries:
   pip install pandas scikit-learn
2. Run the file:
   python fakenews.py
